Vue 学习。
  安装vue。
     cli命令行安装vue脚手架。
       首先全局安装 vue-cli 
         npm install  @vue/cli@3.2.1 -g
       vue 3.0 以上可以使用 vue UI  可视化窗口创建项目。
       创建新项目。
        vue create vuecli3test
            vuecli3test 是项目的名称，项目名称不能以大写字母开头。
